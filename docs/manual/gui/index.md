# GUI reference

Go2Kin runs as a single window with three fixed regions:

- a persistent **[top bar](top-bar.md)** — project, session & participant selection, plus the calibration status indicator;
- **five tabs** — [Live Preview](live-preview.md), [Calibration](calibration-tab.md), [Recording](recording-tab.md), [Processing](processing-tab.md) and [Visualisation](visualisation-tab.md);
- a fixed **[bottom bar](bottom-bar.md)** — camera connection, status and global camera settings.

A typical session moves left to right through the tabs: preview and frame the cameras →
calibrate → record → process → (optionally) visualise.
