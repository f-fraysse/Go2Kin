# First-time setup

Do these steps once per PC/lab. After this, day-to-day work only needs the
[regular workflow](../regular-use.md).

**Prerequisites:** [equipment](../equipment.md) on hand, plus `git` and `conda`
(Miniconda/Anaconda) installed, and a current NVIDIA driver.

1. **[Install Go2Kin](01-installation.md)** — clone the repo and set up the Python environment.
2. **[Create the data folder](02-data-folder.md)** — choose where the (large amount of) data will live.
3. **[Configure your GoPros](03-gopro-configuration.md)** — read each camera's serial number into the config file.
4. **[Connecting the GoPros](04-connecting-gopros.md)** — the connect/disconnect order that keeps USB control reliable.
5. **[First launch of the GUI](05-first-launch.md)** — create your first project, session and participant.
6. **[Intrinsic calibration](06-intrinsic-calibration.md)** — per-camera lens calibration (done rarely).
7. **[Verify your setup](07-verification.md)** — record and process one trial end-to-end.
