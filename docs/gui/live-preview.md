# Tab 1 — Live Preview

Stream a live preview from **one camera at a time**, for positioning and framing.

- Select which camera to preview.
- **Digital zoom** — slider, **+/−** buttons, or direct text entry.
- The preview always runs at **1080p / 30 fps / Linear**, regardless of the recording settings.

!!! warning "Zoom changes invalidate intrinsics"
    Digital zoom changes the effective lens model — if you change it,
    [intrinsic calibration](../first-time-setup/06-intrinsic-calibration.md) must be
    redone for that camera.
