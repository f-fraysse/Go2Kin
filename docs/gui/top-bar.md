## Controls

- **Project / Session / Participant dropdowns** — each with a **+** button to create a new one. Selections persist between launches.
- **Manage** — opens a dialog to view and create subjects: initials, age, sex, **height**, **mass**. Height and mass are required for OpenSim scaling at the processing stage.
- **Calibration status indicator** — **green** = calibrated today, **orange** = older calibration loaded, **red** = no calibration.
