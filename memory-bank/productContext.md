# Product Context

## Why This Project Exists

### Research Problem
Academic and lab researchers need to capture synchronized multi-camera video recordings for motion analysis, behavioral studies, and other research applications.

### Target Users
- Academic researchers
- Lab technicians
- Graduate students conducting video-based research
- Non-technical users who need reliable multi-camera recording

## How It Should Work

### User Experience Goals
1. **Simple Setup**: Connect cameras via USB, launch app, cameras auto-detected
2. **Intuitive Control**: Clear visual status indicators, simple dropdown menus
3. **Reliable Recording**: One-click start/stop for all cameras simultaneously
4. **Organized Output**: Automatic file naming and folder organization
5. **Live Feedback**: Preview capability to verify camera positioning

### Core Workflows

#### Daily Research Session
1. Connect 4 GoPro cameras via USB
2. Launch Go2Kin application
3. Verify all cameras connected (green status indicators)
4. Configure settings if needed (lens, resolution, FPS)
5. Set trial name and output directory
6. Select which cameras to use for this trial
7. Start recording across all selected cameras
8. Stop recording when trial complete
9. Files automatically downloaded and organized
10. Cameras cleaned up (files deleted) for next trial

#### Camera Setup/Verification
1. Use live preview to verify camera positioning
2. Check field of view and focus
3. Adjust physical camera placement as needed
4. Switch between cameras to verify all angles

### Key User Benefits
- **Time Savings**: No manual file management or camera-by-camera operation
- **Reliability**: Synchronized recording prevents data loss
- **Organization**: Consistent file naming and folder structure
- **Simplicity**: No technical expertise required
- **Flexibility**: Support for 1-4 cameras as needed per experiment

## Problems It Solves

### Current Pain Points
- **Manual Coordination**: Starting/stopping multiple cameras individually
- **File Chaos**: Inconsistent naming, scattered files across cameras
- **Setup Time**: Complex configuration for each recording session
- **Sync Issues**: Cameras starting at different times
- **Preview Difficulty**: No easy way to verify camera positioning

### Solution Approach
- **Unified Control**: Single interface for all cameras
- **Automated Workflow**: Minimal user intervention required
- **Smart Organization**: Automatic trial-based file structure
- **Visual Feedback**: Clear status and progress indicators
- **Persistent Settings**: Remember configurations between sessions
